# Game Engine
- JDK Version: Amazon Corretto 17.0.7
- LWJGL Version: 3.3.3
---
My own Game engine capable of 3D and 2D rendering using OpenGL (LWJGL)

### Features:
- Scenes
- Sprite renderers
- Game Objects
- Camera
- Shaders presets and possibility to use own
- Batch Rendering for Performance
- Key input listener
- Mouse input listener
- Demo Scenes
